● PORTFOLIO_V1.log

Este portfólio foi construído com Angular 18 - Standalone, roteamento lazy-loaded, animações nativas e um design system próprio em SCSS.

O Cosmic Theme integra partículas animadas e HUD elements renderizados em tempo real.

O Solar Theme integra um Tema que simula a força de uma Estrela e são renderizados em tempo real.

Projetos Principais

  • Shadow-Flip-Oh! Favicon system dinâmico em Vue.Js
  • Pokedevs Migração HTML → Angular
  • GitHub Stats Node.js + Firebase real-time
